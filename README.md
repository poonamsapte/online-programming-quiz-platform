# online-programming-quiz-platform
Full Stack MERN Project for practicing online programming MCQs.

## Project Overview

The Online Programming Quiz Practice Platform is a full-stack MERN application that allows users to practice programming MCQs across multiple languages such as C, Java, HTML, and Python.
The platform provides an interactive way for students and beginners to test their programming knowledge through quizzes and track their performance.

This project is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** and demonstrates full-stack development including authentication, REST APIs, database management, and dynamic UI rendering.

---

## Features

*  **User Authentication**

  * Secure user registration and login system.

*  **Programming Language Selection**

  * Users can choose a language such as **C, Java, HTML, or Python**.

*  **MCQ Based Quiz System**

  * Multiple choice programming questions.

*  **One Question at a Time**

  * Questions appear sequentially for better focus.

*  **Score Calculation**

  * Final score is displayed after completing the quiz.

*  **Practice-Oriented Platform**

  * Designed to help students prepare for interviews, exams, and programming tests.
